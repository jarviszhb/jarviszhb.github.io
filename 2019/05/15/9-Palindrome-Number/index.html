9.Palindrome Number

Posted on 2019-05-15 | In leetcode

Determine whether an integer is a palindrome. An integer is a palindrome when it reads the same backward as forward.

思路分析

  本题比较简单，可以先判断输入的位数，然后依次判断首位数字是否相等，也可以将输入反转，并判断是否与原输入相等

实现细节

  1. pow()函数的返回值是double型，所以在使用的时候，有时需要对其进行强制类型转换
  2. 在对输入进行反转的过程中，反转后的数，可能会超出int的范围，因此反转变量应该使用long型
