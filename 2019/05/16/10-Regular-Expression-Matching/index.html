10.Regular Expression Matching

Posted on 2019-05-16 | Edited on 2019-06-17 | In leetcode

Given an input string (s) and a pattern (p), implement regular expression matching with support for '.' and ''.
'.' Matches any single character.
'' Matches zero or more of the preceding element.

思路分析

  使用动态规划的思想，状态方程\(dp(i, j)\)表示s[i:]和p[j:]是否匹配，状态转移方程是本题最复杂的部分，当p[j]=='*'时，有两个条件：

  1. dp[i][j+2]，这个条件可以满足所有*表示0个字符的情况
  2. first_match && dp[i+1][j]，这个条件可以满足*表示一个或多个字符的情况

两个条件进行取或操作，即可得出\(dp(i,j)\)的值。

实现细节

  本题有一个非常重要的细节是处理s或p为空的情况：

  1. 当s和p均为空的情况下，返回true
  2. 当p为空，s不为空的情况下，返回false
  3. 当s为空，p不为空的情况下，只有p为形如a*b*c*.*这种每个字母后面都跟随一个*的字符串才返回true，其他情况均返回false

这三种情况正好涉及了\(dp\)函数的初始化问题，从这道题可以看出，在使用动态规划做字符串的问题时，\(dp\)函数的范围应该为字符串的长度加一，这样可以正确的处理边界条件而不用特殊讨论。
