22.Generate Parentheses

Posted on 2019-06-17 | In leetcode

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

思路分析

  这道题的核心代码是ans.push_back("("+left+")"+right)，这句代码可以覆盖所有的情况并且不会产生重复项。做一个简单的证明，设\(A_n=\{n对括号的合法字符串集合\}\)，\(B_n=\{字符串(s_i)s_{n-i-1}的集合\}\)，其中\(s_i\)表示\(A_i\)中的一个元素。 可以证明，任何\(x\in A_n\)，一定满足\(x\in B_n\)，反之也显然成立。

实现细节

  边界条件永远是实现细节里最需要注意的地方，本题在循环中需要将n-1分成两个数，因此n=1不是边界，0可以分解为0和0，只有n=0是边界。
