21.Merge Two Sorted Lists

Posted on 2019-06-16 | In leetcode

Merge two sorted linked lists and return it as a new list. The new list should be made by splicing together the nodes of the first two lists.

思路分析

  判断l1和l2节点值得大小，小的节点加入新链表。

实现细节

  1. 不需要对每一个节点都重新生成，只需要对节点的关系重新拍排列即可，target和l1、l2指针的移动先后顺序十分关键
  2. 循环条件为l1 && l2，加入l1已经为null了，只需要将l2剩余的节点一次性加入链表尾部即可，不需要对每一个节点进行操作
  3. 当需要对剩余节点进行操作时，用l1 || l2，当不需要对剩余节点进行操作时，用l1 && l2
