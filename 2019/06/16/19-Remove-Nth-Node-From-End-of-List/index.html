19.Remove Nth Node From End of List

Posted on 2019-06-16 | In leetcode

Given a linked list, remove the n-th node from the end of list and return its head.

思路分析

  采用curr和target两个指针，并且两个指针始终相差n个节点，因此当curr指针到达最后一个节点的时候，target指针就是要删除节点的上一个节点。

实现细节

  1. 在链表的前面加入一个dummy节点作为头指针，这样做的好处是可以避免一些边界条件的判定。
  2. ListNode* dummy = new ListNode(0);
