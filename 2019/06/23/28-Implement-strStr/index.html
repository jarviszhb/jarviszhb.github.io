Implement strStr(). Return the index of the first occurrence of needle in haystack, or -1 if needle is not part of haystack.

思路分析

  使用KMP算法实现字符串的匹配，将haystack称为目标串，将needle称为模式串。KMP算法的核心思想是，在遍历的过程中r如果不匹配，不对目标串进行回溯，只对模式串进行回溯，而模式串回溯的位置由next数组决定。next数组只与模式串有关，与目标串无关，如，模式串：p = "ababaaabab"，则其next数组的值应为{-1, 0, 0, 1, 2, 3, 1, 1, 2, 3, 4}，也就是说，当目标串的某个字符c与p[5]不匹配的时候，接下来只需要查看c是否与p[next[5]]匹配。

  next数组实际上是储存的是，当前位置之前的子串最长前后缀长度，同样对于字符串p来说，子串ababa的最长前后缀为aba，长度为3，因此next[5]=3，所以next数组的生成方法如下：

  方法2对方法1的一点改进是，再构造next数组的时候，当前位置的字符与欲更新的next位置j的字符是否相同，如果相同，那么在进行匹配的时候，也就不需要单独计算了，因此直接将next[j]赋值给next[i]，可以预见的是，在遇到模式串中有大量重复字符的情况，可以提高KMP算法的效率。

实现细节

  next[0] = -1，这一设置的目的是，当回溯到-1时，代表无法偷懒，应该从模式串的第一位开始从头检查。
